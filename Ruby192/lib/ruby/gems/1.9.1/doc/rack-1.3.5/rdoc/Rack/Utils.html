Rack::Utils contains a grab-bag of useful methods for writing web applications adopted from all kinds of Ruby libraries.

byte_ranges(env, size) click to toggle source

Parses the “Range:” header, if present, into an array of Range objects. Returns nil if the header is missing or syntactically invalid. Returns an empty array if none of the ranges are satisfiable. 

     # File lib/rack/utils.rb, line 285
285:     def byte_ranges(env, size)
286:       # See <http://www.w3.org/Protocols/rfc2616/rfc2616-sec14.html#sec14.35>
287:       http_range = env['HTTP_RANGE']
288:       return nil unless http_range
289:       ranges = []
290:       http_range.split(/,\s*/).each do |range_spec|
291:         matches = range_spec.match(/bytes=(\d*)-(\d*)/)
292:         return nil  unless matches
293:         r0,r1 = matches[1], matches[2]
294:         if r0.empty?
295:           return nil  if r1.empty?
296:           # suffix-byte-range-spec, represents trailing suffix of file
297:           r0 = [size - r1.to_i, 0].max
298:           r1 = size - 1
299:         else
300:           r0 = r0.to_i
301:           if r1.empty?
302:             r1 = size - 1
303:           else
304:             r1 = r1.to_i
305:             return nil  if r1 < r0  # backwards range is syntactically invalid
306:             r1 = size-1  if r1 >= size
307:           end
308:         end
309:         ranges << (r0..r1)  if r0 <= r1
310:       end
311:       ranges
312:     end

rfc2822(time) click to toggle source

Modified version of stdlib time.rb Time#rfc2822 to use ’%d-%b-%Y’ instead of ’% %b %Y’. It assumes that the time is in GMT to comply to the RFC 2109.

NOTE: I’m not sure the RFC says it requires GMT, but is ambigous enough that I’m certain someone implemented only that option. Do not use %a and %b from Time.strptime, it would use localized names for weekday and month.

277:       mon = Time::RFC2822_MONTH_NAME[time.mon - 1]
278:       time.strftime("#{wday}, %d-#{mon}-%Y %H:%M:%S GMT")
279:     end
select_best_encoding(available_encodings, accept_encoding) click to toggle source 
     # File lib/rack/utils.rb, line 164
164:     def select_best_encoding(available_encodings, accept_encoding)
165:       # http://www.w3.org/Protocols/rfc2616/rfc2616-sec14.html
166: 
167:       expanded_accept_encoding =
168:         accept_encoding.map { |m, q|
169:           if m == "*"
170:             (available_encodings - accept_encoding.map { |m2, _| m2 }).map { |m2| [m2, q] }
171:           else
172:             [[m, q]]
173:           end
174:         }.inject([]) { |mem, list|
175:           mem + list
176:         }
177: 
178:       encoding_candidates = expanded_accept_encoding.sort_by { |_, q| -q }.map { |m, _| m }
179: 
180:       unless encoding_candidates.include?("identity")
181:         encoding_candidates.push("identity")
182:       end
183: 
184:       expanded_accept_encoding.find_all { |m, q|
185:         q == 0.0
186:       }.each { |m, _|
187:         encoding_candidates.delete(m)
188:       }
189: 
190:       return (encoding_candidates & available_encodings)[0]
191:     end
set_cookie_header!(header, key, value) click to toggle source 
     # File lib/rack/utils.rb, line 194
194:     def set_cookie_header!(header, key, value)
195:       case value
196:       when Hash
197:         domain  = "; domain="  + value[:domain] if value[:domain]
198:         path    = "; path="    + value[:path]   if value[:path]
199:         # According to RFC 2109, we need dashes here.
200:         # N.B.: cgi.rb uses spaces...
201:         expires = "; expires=" +
202:           rfc2822(value[:expires].clone.gmtime) if value[:expires]
203:         secure = "; secure"  if value[:secure]
204:         httponly = "; HttpOnly" if value[:httponly]
205:         value = value[:value]
206:       end
207:       value = [value] unless Array === value
208:       cookie = escape(key) + "=" +
209:         value.map { |v| escape v }.join("&") +
210:         "#{domain}#{path}#{expires}#{secure}#{httponly}"
211: 
212:       case header["Set-Cookie"]
213:       when nil, ''
214:         header["Set-Cookie"] = cookie
215:       when String
216:         header["Set-Cookie"] = [header["Set-Cookie"], cookie].join("\n")
217:       when Array
218:         header["Set-Cookie"] = (header["Set-Cookie"] + [cookie]).join("\n")
219:       end
220: 
221:       nil
222:     end
